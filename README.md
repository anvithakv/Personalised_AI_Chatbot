# Personal AI Mentor

## Overview

Personal AI Mentor is a Python-based AI learning assistant powered by Google's Gemini API. It creates personalized learning profiles and provides customized responses based on the user's learning interests, current skill level, and goals.

The system acts as a virtual mentor that remembers user preferences and stores chat history for future reference.

---

## Features

### User Profile Management

* Create multiple learner profiles
* Store learning interests
* Store skill levels (Beginner, Intermediate, Advanced)
* Store learning goals
* Update profile information anytime

### AI-Powered Learning Assistance

* Gemini AI integration
* Personalized responses
* Level-based explanations
* Interactive learning support

### Chat History

* Automatically saves conversations
* Stores timestamps
* View previous discussions
* Separate history for each profile

### Profile Switching

* Manage multiple users
* Switch between profiles instantly

### Data Storage

* JSON-based profile storage
* JSON-based chat history storage
* Lightweight and easy to manage

## Available Commands

| Command       | Description               |
| ------------- | ------------------------- |
| history       | View previous chats       |
| profile       | View current profile      |
| updateprofile | Update profile details    |
| newprofile    | Create a new profile      |
| switch        | Switch to another profile |
| exit          | Exit the application      |

---

## Example Workflow

1. Create a learning profile.
2. Specify learning topic.
3. Choose skill level.
4. Set learning goal.
5. Ask questions to the AI mentor.
6. Receive personalized responses.
7. View saved chat history anytime.
